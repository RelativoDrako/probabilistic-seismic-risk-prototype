# Architecture Support

This folder contains **supporting architectural material** for PSRP.

Its role is to complement `docs/architecture.md`, not compete with it.

Use this folder for:

- system-surface diagrams;
- data-flow diagrams;
- deployment-boundary visuals;
- compact architecture support notes.

## Reading rule

- `docs/architecture.md` = canonical architectural explanation
- `architecture/` = visual and diagrammatic support layer
