# Executive Controlled Review

This folder contains the controlled public-review storyboard for PSRP.

It is not a mockup gallery. It is a lightweight validation aid that compares the real running web surface against the intended publication narrative.

Use it after launching the local stack and confirming that the Streamlit surface loads correctly.

## What it validates

- Executive Overview readability.
- Mexico Executive Map as the main geographic narrative.
- Recorded-events and tectonic-context maps as separate supporting views.
- Regional Risk View as the selected-region drill-down.
- Model Evaluation as bounded analytics, not overclaimed science.
- Methodology, API posture, and future projection as publication support.
