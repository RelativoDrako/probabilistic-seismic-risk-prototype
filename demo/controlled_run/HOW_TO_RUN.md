# Demo Validation

## Objective

Use this folder to confirm that screenshots, captions, and executive review materials still match the real PSRP web surface.

## Review sequence

1. Start the PSRP local API and web surface.
2. Open `http://127.0.0.1:8501`.
3. Validate the main sections in the live site.
4. Open `demo/controlled_run/visual_review_index.html`.
5. Compare each storyboard panel with the corresponding live section.
6. Regenerate screenshots if the live site changed materially.

## Required checks

- screenshots correspond to real sections;
- captions describe current behavior, not desired future behavior;
- no premium mockups are used;
- maps remain primary visual evidence;
- paths are relative and publication-safe;
- model and future-work claims remain bounded.
