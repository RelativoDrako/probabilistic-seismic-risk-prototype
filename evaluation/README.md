# Evaluation

## Purpose

This folder explains what the prototype evaluates today, what it does not evaluate, and how to read the metrics and scenarios without overstating analytical scope.

## What evaluation means here

Evaluation in PSRP is used to verify:

- bounded analytical coherence;
- interpretability of outputs;
- repeatability of reports and artifacts;
- publication-safe communication of results.

It does **not** imply real-time operational performance, deterministic forecasting, or production readiness.
