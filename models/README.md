# Models

## Purpose

This folder explains the current model posture of PSRP.

The prototype starts with **baseline models first** because they are easier to interpret, easier to review, and professionally safer than jumping directly into complexity.

Use this folder to understand:

- why baseline models are the current default;
- how experimental models should be interpreted;
- how the model layer relates to the bounded thesis of the repository.

See `docs/model_card.md` for the canonical analytical boundary.
